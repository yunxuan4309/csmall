## 四、访问地址规划

### 备案前（IP 访问）

```
前台首页：http://公网IP/
API 网关：http://公网IP:10087/  （Nginx 代理后不需要加端口）
```

### 备案后（域名访问）

```
前台首页：http://coolshark-shop.cn/
API 网关：http://coolshark-shop.cn/ （Nginx 反向代理到 Gateway）
```

### HTTPS 上线后

```
前台首页：https://coolshark-shop.cn/
HTTP 自动跳转 HTTPS
```

## 七、敏感信息说明

以下信息**不写入此文档**，通过环境变量或 Nacos 配置中心注入：

- 服务器公网/内网 IP → 环境变量 `ALIYUN_SERVER_IP`
- MySQL 用户名/密码 → 环境变量 `MYSQL_USERNAME` / `MYSQL_PASSWORD`
- Redis 密码 → 环境变量 `REDIS_PASSWORD`
- RabbitMQ 用户名/密码 → 环境变量 `RABBITMQ_USERNAME` / `RABBITMQ_PASSWORD`
- JWT Secret → 环境变量或 Nacos
- 文件资源访问 URL → 环境变量 `RESOURCE_HOST`
- 服务器 SSH 密码 → 仅保存在本地密码管理器中

> 这些信息不应出现在 Git 仓库中，包括私有仓库。
