## 模块依赖关系

### 基础服务模块
- **mall-gateway-server**
  - 依赖: mall-common

### 后台管理模块 (mall-ams)
- **mall-ams-webapi**
  - 依赖: mall-ams-service, mall-pojo, mall-common

### 用户管理模块 (mall-ums)
- **mall-ums-webapi**
  - 依赖: mall-ums-service, mall-common

### 商品管理模块 (mall-product)
- **mall-product-webapi**
  - 依赖: mall-product-service, mall-pojo, mall-common

### 订单管理模块 (mall-order)
- **mall-order-webapi**
  - 依赖: mall-order-service, mall-product-service, mall-common

### 秒杀模块 (mall-seckill)
- **mall-seckill-webapi**
  - 依赖: mall-seckill-service, mall-order-service, mall-ums-service, mall-product-service, mall-common

### 搜索模块 (mall-search)
- **mall-search-webapi**
  - 依赖: mall-search-service, mall-product-service

### 前端服务模块 (mall-front)
- **mall-front-webapi**
  - 依赖: mall-front-service, mall-product-service, mall-common

## 公共依赖
所有WebAPI模块均依赖以下公共组件：
- mall-common
- Spring Boot Web
- Spring Security
- JJWT
- Lombok
- Knife4j
- Fastjson
